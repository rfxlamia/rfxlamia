### 🧠 What I'm actually doing

- **Building** [`Verminal`](https://github.com/rfxlamia/verminal) — a tiling terminal emulator for AI multi-agent workflows. Electron + Svelte 5 + node-pty. Replacing Warp, for myself, by myself.
- **Maintaining** [`skillkit`](https://github.com/rfxlamia/skillkit) — a framework for packaging reusable AI skills. 93 ⭐, page-one on Google, and I still don't fully believe that happened.
- **Shipping** FSTrack & WooMaps at GGF — agricultural ops systems running in the field, not just on localhost.
- **Researching** how agents can collision-detect each other's architectural decisions (ACCORD). Still experimental, but it's the idea I care about most right now.
